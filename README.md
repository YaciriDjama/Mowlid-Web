# Bright Photo Booth Website

A modern black-and-blue photo booth website built for bookings, events, and gallery showcases.

## Features

- Responsive single-page layout
- Hero section with booking call-to-action
- Events and services section
- Interactive image carousel with thumbnails
- Contact links for email and phone
